# EcommerceApp
An e-commerce mobile app for selling and/or buying smartphone.   
Language: Kotlin
Technologies:

  - MVVM architecture
  
  - Android Jetpack (Livedata, view binding, databinding...)
  
  - Kotlin Coroutine
  
  - Retrofit/OkHttp
